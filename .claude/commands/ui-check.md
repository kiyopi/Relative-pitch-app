# UI Component Check

このコマンドはUIコンポーネントの実装状況と一貫性をチェックします。

## Instructions

以下の手順でUIコンポーネントの状況を確認してください：

1. **ui-catalog-essentials.html**を読み込み、利用可能なコンポーネントを確認
2. **base.css**で定義されているユーティリティクラスを確認
3. **インライン記述**がないかチェック（style属性の使用状況）
4. **プログレスバー**の実装が統一されているか確認
5. **音量バー制御**が適切に実装されているか確認

問題が見つかった場合は具体的な修正提案を含めて報告してください。
