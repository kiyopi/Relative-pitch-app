# Claude Ask

プロジェクトに関する質問や疑問を解決するためのコマンドです。

## Instructions

ユーザーからの質問に対して、以下の手順で回答してください：

### 1. 質問の分類
質問を以下のカテゴリに分類し、適切なアプローチを選択：
- **技術実装**: コード、ライブラリ、アーキテクチャ関連
- **設計原則**: 相対音感原則、UI設計、プロジェクト方針
- **プロジェクト状況**: 進捗、ファイル構成、実装状況
- **トラブルシューティング**: エラー、問題、解決方法

### 2. 関連リソース確認
質問に関連する以下のファイルを確認：
- `CLAUDE.md` - 開発ガイドライン
- `CONTEXT.md` - プロジェクト概要
- `APP_SPECIFICATION.md` - アプリ仕様
- 関連する実装ファイル

### 3. 回答形式
- **簡潔で具体的な回答**を提供
- **実装例やコード例**があれば含める
- **関連ドキュメント**への参照を含める
- **注意点やベストプラクティス**を含める

### 4. プロジェクト固有の配慮
- 相対音感原則（音名・周波数表示禁止）を考慮
- PitchProライブラリの正しい使用方法
- UIコンポーネントの統一性
- インライン記述禁止原則

質問が不明確な場合は、より具体的な情報を求めてから回答してください。
