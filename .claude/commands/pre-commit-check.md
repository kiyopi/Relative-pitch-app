# Pre-commit Check

コミット前に実行する包括的なチェックコマンドです。

## Instructions

コミット前に以下の項目を順次チェックし、問題があれば修正してからコミットしてください：

### 1. Git Status確認
- 変更ファイルの一覧を確認
- 意図しないファイルが含まれていないかチェック
- コミットメッセージの準備

### 2. コード品質チェック
- HTMLファイルでインライン記述（style属性）がないか確認
- JavaScript でインラインCSSがないか確認  
- ユーティリティクラス（base.css）が適切に使用されているか確認

### 3. プロジェクト固有ルール確認
- 相対音感原則：トレーニング中の音名・周波数表示禁止が守られているか
- 音量値取得：result.volumeコールバック方式が使用されているか
- VolumeBarController：統一された音量バー制御が使用されているか

### 4. 動作確認
- 主要機能が動作するか簡易テスト
- コンソールエラーがないか確認

### 5. ドキュメント更新
- CLAUDE.mdの作業状況を更新する必要があるか確認
- 新機能があればドキュメント追加

すべてクリアしたら適切なコミットメッセージでコミットを実行してください。
