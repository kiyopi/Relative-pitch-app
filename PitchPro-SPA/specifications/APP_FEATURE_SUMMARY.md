# 8va相対音感トレーニングアプリ - 特徴要約レポート

## 1. アプリの核心的価値 (Core Value)
このアプリは単なる音楽トレーニングツールではなく、**「脳内ピアノ理論」に基づいたコミュニケーション能力向上ツール**としての側面も持ち合わせています。

### 脳内ピアノ理論 (Brain Piano Theory)
*   **音声と音楽の等価性**: 人の声はフーリエ解析するとピアノの音と同じ成分で構成されており、人は「脳内にピアノを持っている」と言えます。
*   **人間関係は12音律**: 人と人のコミュニケーションは音（言葉）で行われ、その関係性は12音律で説明できます。
*   **相対音感 = コミュニケーション力**: 相対音感を鍛えることは、単に歌がうまくなるだけでなく、相手の「音（意図・感情）」を正確に捉え、適切な「音（言葉）」を返す能力（＝円滑な人間関係）に直結します。

## 2. トレーニングの原則 (Core Principles)
*   **絶対音感の排除**: トレーニング中は基音の音名（C3など）や周波数を一切表示しません。これにより、絶対音感に頼らず、純粋に「音の相対的な距離」を感じ取る能力を養います。
*   **統合進行ガイド**: 視覚的なガイド（ドレミ表示）と聴覚的なガイド（基音再生）を一体化させ、直感的なトレーニングを提供します。

## 3. トレーニングモード (Training Modes)
段階的な学習システムを採用しています。

1.  **ランダム基音モード (初級)**
    *   **目的**: 基礎的な相対音感の習得。
    *   **内容**: ランダムに選ばれた基音に対し、ドレミファソラシド（8音）を歌う。
    *   **特徴**: 1セッション完結型。

2.  **連続チャレンジモード (中級)**
    *   **目的**: 集中力と瞬発力の強化。
    *   **内容**: 休憩なしで8セッション連続で行う。
    *   **特徴**: 半音を含む12音階から基音が選ばれる。

3.  **12音階モード (上級)**
    *   **目的**: 完全な相対音感の習得。
    *   **内容**: クロマチック（半音階）全てを網羅する。
    *   **要件**: 2.0オクターブ以上の音域が必要。

## 4. 独自の機能 (Unique Features)
*   **脳内処理メカニズムへの対応**:
    *   **左脳処理音 (7音)**: C, C#, D, D#, E, F, F# （言語的・論理的処理）
    *   **両脳処理音 (5音)**: G, G#, A, B♭, B （空間的・感覚的処理、習得難易度が高い）
    *   これらを意識したトレーニングにより、脳の両半球をバランスよく鍛えます。

*   **ダイレクトアクセス & マイクテスト**:
    *   URL共有から即座にトレーニングを開始できるスムーズな導線。
    *   必要な時だけマイク許可を求めるユーザーフレンドリーな設計。

## 5. オンボーディングへの応用案
このアプリの独自性（特に「脳内ピアノ理論」と「コミュニケーションへの効果」）は、一般的な音楽アプリとは一線を画すため、初回起動時のオンボーディングでユーザーに強く印象付ける必要があります。

**スライド構成案:**
1.  **Welcome**: 「あなたの脳内にはピアノがある」という驚きのある導入。
2.  **Theory**: 「相対音感」が「人間関係」を変えるというメリットの提示。
3.  **Method**: 「絶対音感に頼らない」独自のトレーニング方法の解説。
4.  **Action**: さあ、あなたの脳内ピアノを調律しましょう。
