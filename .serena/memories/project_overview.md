# 8va相対音感トレーニングアプリ - プロジェクト概要

## プロジェクトの目的
相対音感（基準音に対する音程関係を聞き分ける能力）を効果的に鍛えるWebアプリケーション。
歌唱力向上だけでなく、脳内ピアノ理論に基づきコミュニケーション能力向上にも寄与。

## アプリの3つのトレーニングモード
1. **ランダム基音モード（初級）**: 8セッション、C3オクターブ8音からランダム基音
2. **連続チャレンジモード（中級）**: 12セッション連続、クロマチック12音使用
3. **12音階モード（上級）**: 12-24セッション、全半音順次使用でS級判定

## ターゲットユーザー
- 音楽学習者（初級〜中級）
- カラオケ愛好家、合唱団メンバー
- コミュニケーション能力向上を目指す一般ユーザー

## 最重要設計原則
- **相対音感原則**: トレーニング中の基音音名・周波数表示は完全禁止
- **音域適応**: ユーザーの快適音域に合わせた基音選択
- **リアルタイム判定**: ±50セント以内で合格判定

## 技術的特徴
- **PWA対応**: オフライン動作、モバイル最適化
- **音声処理**: Web Audio API + Pitchy v4で高精度音程検出
- **デバイス最適化**: PC/iPhone/iPad別の感度設定
