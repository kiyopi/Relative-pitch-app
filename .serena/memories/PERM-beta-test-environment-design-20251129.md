## 案内ページ構成

| セクション | 内容 |
|-----------|------|
| ヘッダー | 「8va βテスト版」ロゴ・バージョン表示 |
| 注意事項 | βテストであること、データ消失の可能性 |
| 動作環境 | 推奨ブラウザ（Safari/Chrome）、マイク必須 |
| テスト項目 | 重点的に確認してほしいポイント |
| 開始ボタン | アプリへのリンク |
| フィードバック | Googleフォームへのリンク |

---

## フィードバックフォーム項目

### 基本情報
1. お名前/ニックネーム（任意）
2. 使用デバイス
   - iPhone / iPad / Android / PC（Mac） / PC（Windows）
3. 使用ブラウザ
   - Safari / Chrome / その他
4. イヤホン/ヘッドホン使用
   - 有線イヤホン / スピーカー（イヤホンなし） / Bluetooth（非推奨）

### テスト内容
5. テストしたモード（複数選択可）
   - ランダム基音 / 連続チャレンジ / 12音階
6. テストした方向
   - 上行のみ / 下行のみ / 両方

### 音声・マイク関連
7. 基音（ピアノ音）の音量
   - 大きすぎる / ちょうど良い / 小さすぎる / 聞こえない
8. マイクの感度（声の拾い具合）
   - 敏感すぎる（雑音を拾う） / ちょうど良い / 鈍すぎる（声を拾わない）
9. 音量バーの反応
   - スムーズ / やや遅い / 動かない・不安定
10. 音域テストの精度
    - 正確に測定できた / やや不正確 / 全く測定できなかった

### 評価（5段階）
11. 全体的な使いやすさ
12. 音程測定の精度
13. 画面の見やすさ
14. 操作の分かりやすさ

### 自由記述
15. 良かった点
16. 改善してほしい点
17. バグ・エラーの報告（スクリーンショット添付可）
18. その他コメント

## Googleフォーム作成手順

1. Google Forms (https://forms.google.com/) にアクセス
2. 新規フォーム作成「8va βテスト フィードバック」
3. 上記項目を順番に追加
4. 回答の通知設定をON
5. フォームURLを取得
6. 案内ページにリンク埋め込み
